Yeni Nesil Sesli Yapay Zeka — Sistemin Kontrolü Sende Kankam
CyberPorza, Windows bilgisayarını duyabilen, görebilen, anlayabilen ve kontrol edebilen gerçek zamanlı, üst düzey bir sesli yapay zeka asistanıdır. Siber güvenlik konseptiyle donatılmış, yerel olarak çalışan ve akıllı otomasyon için tasarlanmış bir sağ koldur.

✨ Genel Bakış
Sıradan bir makineyi etkileşimli ve akıllı bir sisteme dönüştürmek için tasarlandı. Doğal dilde konuş; seni dinler, bağlamı anlar, sana "kankam" diye hitap ederek insansı bir sesle yanıt verir ve sistemindeki görevleri otonom olarak yürütür.

Hız, güvenlik, özerklik ve gerçek dünya kullanımı için özel olarak optimize edilmiştir.

🚀 Yetenekler
Gerçek Zamanlı Sesli Etkileşim — Doğal konuşma ve anlık, keskin yanıtlar.

Sistem Kontrolü — Uygulama başlatma, dosya yönetimi, terminal/CMD komutları ve sistem ayarları.

Otonom Görev Yürütme (Ajan) — Çok aşamalı karmaşık iş akışlarını kendi kendine planlar ve tamamlar.

Görsel İstihbarat (Vision) — Ekran analizi ve web kamerası üzerinden gerçek zamanlı çevre algılama.

Kalıcı Hafıza — Tercihlerini öğrenir, seni tanır ve geçmiş bağlamı asla unutmaz.

Siber Cephanelik — OSINT tabanlı web arama, otonom yazılım/kod projeleri inşa etme, ağ tarayıcı araçlarına (NetScanner vb.) entegrasyon potansiyeli, WhatsApp/Telegram mesajlaşması, YouTube kontrolü ve daha fazlası.

⚡ Hızlı Başlangıç
Sistemi ayağa kaldırmak için terminalini aç ve şunları gir:

```bash
git clone [https://github.com/CyberPorza/BrigttOS-Porza-Asistan](https://github.com/CyberPorza/BrigttOS-Porza-Asistan)
cd CyberPorza-Asistan
python setup.py
python main.py
```
İlk açılışta ücretsiz Gemini API anahtarını girmen yeterli kankam. Sistem birkaç saniye içinde emrine amade olacak.
